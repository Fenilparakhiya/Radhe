# Radhe Pani Puri Website

Static Next.js 14 restaurant website for Radhe Pani Puri.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Important edits before deployment
- Replace `siteUrl` in `data/business.ts` with your real domain.
- Add real food photos later if you want a stronger gallery.
